## Prerequisites

- **Node.js 24+**
- **Yarn 4** (managed via corepack)

## Getting Started

```bash
# Clone the repository
git clone https://github.com/abinnovision/react-commons.git
cd react-commons

# Enable corepack (provides Yarn 4)
corepack enable

# Install dependencies
yarn install

# Build all packages
yarn build
```

## Development Mode

Start all packages in watch mode:

```bash
yarn start:dev
```

This runs in parallel:
- `tsdown --watch` for library packages
- Storybook dev server on port 6006
- Docs site dev server

## Common Commands

| Command | Description |
|---------|-------------|
| `yarn build` | Build all packages |
| `yarn start:dev` | Start dev mode (watch + Storybook + docs) |
| `yarn fix` | Auto-fix linting and formatting |
| `yarn check` | Check linting and formatting |
| `yarn test-unit` | Run all unit tests |
| `yarn clean` | Clean build artifacts |

## Working with Specific Packages

```bash
# Build a single package
yarn turbo build --filter=react-overlay-stack

# Run tests for a single package
yarn turbo test-unit --filter=react-overlay-stack

# Start Storybook
yarn workspace @internal/examples dev
```
