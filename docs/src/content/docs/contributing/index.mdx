## Commit Convention

This project uses [Conventional Commits](https://www.conventionalcommits.org/). Commit messages are validated by commitlint via a git hook.

```
feat(overlay-stack): add closeAll method
fix(docs): correct API reference link
chore: update dependencies
```

## Pull Requests

- All PRs target `main`
- CI runs build, lint, format checks, and unit tests
- Commits must follow the conventional commit format
