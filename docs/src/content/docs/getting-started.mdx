## Prerequisites

- Node.js 24+
- Yarn 4

## Available Packages

| Package | Description |
|---------|-------------|
| [`react-overlay-stack`](/packages/react-overlay-stack/) | Type-safe, unstyled overlay manager for React 19 |

## Installation

Each package is published independently to npm. Install the package you need:

```bash
npm install react-overlay-stack
# or
yarn add react-overlay-stack
```

See the individual package documentation for detailed setup instructions.
